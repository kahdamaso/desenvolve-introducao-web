Exercícios - Hello World

1. Cite as diferenças entre os elementos "head" e "body" de uma página HTML. Quando deve ser usado cada um?

O elemento "head" é a parte da página que não aparece para quem está acessando o site. Ele serve para guardar informações importantes para o navegador, como o título da página, configurações e outras informações sobre o site. O "head" deve ser usado quando a informação não precisa aparecer na tela, mas é necessária para o funcionamento da página.

O elemento "body" é a parte da página que aparece para o usuário. Tudo o que a pessoa vê no site fica dentro do "body", como textos, títulos, imagens e parágrafos. O "body" deve ser usado sempre que o conteúdo for visível ou puder ser lido pelo usuário.
