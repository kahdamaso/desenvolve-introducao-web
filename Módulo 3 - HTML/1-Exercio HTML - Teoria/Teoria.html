Atividade de HTML

1. Função das tags, corpo e atributos de um elemento HTML

As tags são como comandos que avisam ao navegador o que aquele pedaço do código representa. Por exemplo, se eu usar a tag <p>, eu estou dizendo: “isso aqui é um parágrafo”.

O corpo é o que está dentro da tag, ou seja, é o conteúdo que vai aparecer na tela, como um texto, um título ou algo do tipo.

Já os atributos são informações extras que eu posso colocar na tag para mudar alguma coisa, como a imagem que quero mostrar, a cor de um botão ou o link que vai ser aberto quando alguém clicar.

2. O que é um elemento vazio? Qual a sua funcionalidade?

Um elemento vazio é uma tag que não tem conteúdo dentro dela, ela já faz sua função sozinha. A gente não precisa escrever nada entre a abertura e o fechamento, porque ela não tem fechamento.

Exemplos:

  • A tag <br> serve para quebrar a linha, como se fosse um “enter”.
  • A tag <img> é usada para mostrar uma imagem, e dentro dela a gente coloca o endereço da imagem com um atributo.
